➕ Ajouter un nouvel Agent ou Superviseur

Pourquoi via Firebase Console ? C'est la méthode la plus sécurisée. Elle évite les conflits de session et garantit que chaque compte a un rôle clair.

📋 Étapes (30 secondes)

  1. Ouvre 👉 Firebase Auth → Utilisateurs
  2. Clique sur "Ajouter un utilisateur"
  3. Email : id@millesservices.tg (ex: agt002@millesservices.tg)
  4. Mot de passe : Temp2026! (l'agent le changera plus tard)
  5. Clique sur "Ajouter" → Copie l'UID affiché
  6. Ouvre 👉 Realtime Database → Données
  7. Dans users/, clique sur + → Clé : l'UID copié
  8. Ajoute les champs :
    role : "agent" (ou "superviseur")
    name : "Nom Prénom"
    site : "Site choisi" (agent uniquement)
